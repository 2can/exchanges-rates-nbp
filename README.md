# Exchanges::Rates::Nbp

Simbple gem that gets exchange rates from the Polish National Bank.

## Installation

Add this line to your application's Gemfile:

    gem 'exchanges-rates-nbp'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exchanges-rates-nbp
